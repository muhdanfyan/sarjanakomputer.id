# Jasa Automasi Perusahaan + AI

## Konsep
Membantu perusahaan mengotomasi kegiatan operasional menjadi lebih simpel dengan bantuan AI.

## Level Layanan

| Level | Model | Estimasi Harga | Cocok Untuk |
|-------|-------|---------------|-------------|
| 🥉 **Otomasi Dasar** | Build sekali + maintenance ringan | 3-5jt build + 500rb/bln | UMKM, toko kecil |
| 🥈 **Otomasi Bisnis** | Build + maintain bulanan | 8-15jt + 1-2jt/bln | Perusahaan menengah |
| 🥇 **Enterprise AI** | Konsultasi + build + maintain | Custom (30jt++) | Pemerintah/korporasi |

## Contoh Layanan Automasi

### Administratif (mudah, cepat deliver)
- Absensi digital (QR code / fingerprint)
- Surat menyurat digital (template otomatis)
- Pengarsipan dokumen (scan → kategori → search)
- Approval workflow (cuti, reimbursement, pengajuan)

### Operasional (sedang)
- Laporan keuangan otomatis (dari transaksi harian → laporan bulanan)
- Inventory management (stok update otomatis, reorder alert)
- Customer follow-up (WA/email otomatis)
- Scheduling & jadwal

### AI-Powered (advanced)
- **AI Chatbot** untuk customer service 24 jam
- **AI Document Processing** — scan KTP/NPWP → auto-fill form
- **AI Report Generator** — dari data mentah jadi laporan berbahasa Indonesia
- **AI Recommendation** — rekomendasi berdasarkan data historis
