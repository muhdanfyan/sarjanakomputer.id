## 📐 Detail Tiap Subdomain

### 1. profil.sarjanakomputer.id
**Tujuan:** Company profile resmi — kesan profesional & trustworthy
**Konten:**
- Hero: Nama perusahaan + tagline + CTA "Hubungi Kami"
- Tentang: Sejarah, visi, misi, nilai perusahaan
- Struktur Organisasi: Bagan lengkap dengan foto personil
- Legalitas: Akta pendirian, NIB, NPWP, SIUP, dll (scan dokumen)
- Klien: Logo klien yang pernah bekerja sama
- Testimoni: Kata-kata klien
- Portofolio: Grid proyek unggulan
- Kontak: Alamat kantor (maps embed), telepon, email, form kontak
- Footer: Copyright, sosial media

### 2. aplikasi.sarjanakomputer.id
**Tujuan:** Landing page jasa pembuatan aplikasi — meyakinkan calon klien
**Konten:**
- Hero: "Buat Aplikasi Impian Anda" + CTA
- Layanan: Breakdown jasa (Web, Mobile Android, Desktop, Custom Software)
- Proses: Bagaimana cara kerja (4-5 langkah: Konsultasi → Desain → Develop → Launch → Maintenance)
- Paket Harga: 3 paket (Basic, Business, Enterprise) atau custom quote
- Portofolio: Proyek yang relevan (Sipanda, SIMKES, E-Retribusi, dll)
- Testimoni: Bukti sosial
- FAQ: Pertanyaan umum seputar pembuatan aplikasi
- Kontak: Form order + WhatsApp button

### 3. automasi.sarjanakomputer.id
**Tujuan:** Landing page jasa automasi + AI — solusi efisiensi bisnis
**Konten:**
- Hero: "Automasi Bisnis Anda dengan AI" + CTA
- Masalah: Kenapa perlu automasi? (data ngantri, kerja manual, human error)
- Solusi: Layanan yang ditawarkan
  - Otomasi Administratif (absensi, surat, arsip digital)
  - Otomasi Operasional (inventory, report, scheduling)
  - AI-Powered (chatbot, document processing, AI report)
- Paket Harga: 3 tier (Bronze/Silver/Gold)
- Case Study: Contoh implementasi
- FAQ: Seputar automasi
- Kontak: Konsultasi gratis

### 4. academy.sarjanakomputer.id
**Tujuan:** Sarjana Komputer Academy — pusat pendidikan & pelatihan IT
**Konten:**
- Hero: "Jadi Sarjana Komputer Profesional" + CTA
- Program:
  - 🎓 Bootcamp (Fullstack, Backend, Frontend, Android)
  - 📝 Bimbingan Skripsi/Tugas Akhir
  - 💼 Magang / Project-Based Learning
  - 🎤 Webinar & Workshop
- Mentor: Perkenalan tim pengajar (Muhdan, Hayat, Arif, dll)
- Jadwal: Kalender program & pendaftaran
- Testimoni: Alumni / peserta sebelumnya
- Pricing: Biaya per program
- Pendaftaran: Form register + payment gateway

### 5. blog.sarjanakomputer.id
**Tujuan:** Tech News — update teknologi dari luar negeri + artikel informatif
**Konten:**
- Artikel terbaru (grid/list view)
- Kategori: AI, Web Dev, Mobile, Cybersecurity, Cloud, Startup

## 🛠️ Teknis Setup Subdomain

Setiap subdomain akan:
1. **Folder:** `/html/{subdomain}/` di repo yang sama (atau repo terpisah untuk yang kompleks)
2. **Vercel:** Tambah domain di project Vercel (`skomindo`) — Vercel handle routing
3. **Cloudflare:** Tambah DNS record `CNAME {subdomain} → cname.vercel-dns.com` (proxy ON)
4. **Desain:** Bootstrap 5 + AOS (konsisten dengan main site)
5. **Konten:** Bahasa Indonesia, profesional, mobile-first

Contoh struktur folder:
```
sarjanakomputer.id/
├── html/
│   ├── index.html                 ← Main landing
│   ├── struktur-organisasi.html
│   ├── images/
│   ├── css/
│   ├── js/
│   ├── profil/                    ← profil.sarjanakomputer.id
│   │   └── index.html
│   ├── aplikasi/                  ← aplikasi.sarjanakomputer.id
│   │   └── index.html
│   ├── automasi/                  ← automasi.sarjanakomputer.id
│   │   └── index.html
│   └── academy/                   ← academy.sarjanakomputer.id
│       └── index.html
├── knowledge/
│   └── subdomain-plan.md          ← Dokumen ini
└── vercel.json
```
