# Sarjana Komputer Academy — Rencana

> Wadah pendidikan untuk calon-calon sarjana komputer.

## Target Audiens
- Mahasiswa IT aktif (semester 4-8)
- Fresh graduate yang ingin upgrade skill
- Umum (untuk kelas tertentu)

## Program

### A. Bootcamp & Pelatihan
| Program | Durasi | Target | Output |
|---------|--------|--------|--------|
| Full-stack Web Dev | 3 bulan | Mahasiswa tingkat akhir | Bisa bikin web sendiri |
| Backend (Go/Node) | 2 bulan | Yang mau fokus backend | Siap kerja sebagai backend dev |
| Frontend (React) | 2 bulan | Yang suka desain & UI | Portfolio website |
| Android Basic | 2 bulan | Pemula | Aplikasi android sederhana |
| UI/UX Design | 1 bulan | Desainer pemula | Portfolio Figma |

### B. Bimbingan Skripsi/Tugas Akhir
- Konsultasi topik: "Apa yang bisa saya buat untuk skripsi?"
- Bimbingan teknis: Dibimbing coding sampai jadi
- Bantuan dokumen: BAB 1-5 dengan pendekatan teknis yang benar
- Output: Aplikasi jadi + laporan skripsi siap sidang

### C. Magang / Project-Based Learning
Mahasiswa magang → ikut ngerjain project real (Sipanda, SIMKES, dll) → dapat pengalaman + sertifikat.

### D. Komunitas & Networking
- Forum diskusi, study group, webinar
- Koneksi ke alumni yang sudah bekerja di industri
- Info lowongan kerja IT

## Model
- **Lokal (Baubau):** Kelas tatap muka
- **Online:** Via Zoom/Discord untuk nasional
- **Hybrid:** Kombinasi keduanya
