# Tech News — Blog

## Konsep
Blog teknologi di halaman utama `sarjanakomputer.id` yang berisi berita-berita terbaru teknologi dari luar negeri.

## Tujuan
Menginformasikan update teknologi terbaru yang dibutuhkan oleh para sarjana komputer agar bisa bersaing di industri.

## Kategori Konten
- AI / Machine Learning
- Web Development
- Mobile Development
- Cybersecurity
- Cloud Computing
- Startup & Tech Industry
- Open Source

## Model Konten
- **Auto-fetch:** RSS aggregator dari TechCrunch, The Verge, ArsTechnica, Hacker News
- **Manual:** Saduran/terjemahan + opini lokal oleh tim
- **Campuran:** Auto-fetch untuk headline, manual untuk artikel opini

## Layout di Homepage
- Grid card di bawah hero section atau di bawah layanan
- Thumbnail + judul + kategori + tanggal + excerpt
- 3-6 artikel terbaru di homepage
- Full list di halaman `/blog`
