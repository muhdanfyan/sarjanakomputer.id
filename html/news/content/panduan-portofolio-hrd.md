Portofolio adalah **senjata utama** fresh graduate IT. IPK 4.0 tanpa portofolio kalah saing sama IPK 3.0 dengan portofolio mentereng. Kenapa? Karena portofolio membuktikan **kamu bisa bikin sesuatu**.

## Platform Portofolio

### GitHub — Wajib!
Profil GitHub yang rapi adalah portofolio paling kredibel. Pastikan:
- **Foto profil** yang profesional
- **README profil** dengan bio singkat, skill, dan kontak
- **Pinned repositories** — 3-6 proyek terbaikmu
- **Commit streak** — menunjukkan konsistensi
- **Deskripsi repo** yang jelas (README setiap proyek)

### Personal Website
Buat website portofolio sendiri untuk showcase yang lebih detail. Tech stack rekomendasi:
- **Next.js / Astro** — static site, performa tinggi
- **Tailwind CSS** — styling cepat dan responsif
- Deploy gratis di **Vercel / Netlify**

### Platform Lain
- **LinkedIn** — update profil dengan proyek dan sertifikasi
- **Dribbble / Behance** — untuk UI/UX designer
- **Kaggle** — untuk data scientist

## Isi Portofolio yang Bikin HRD Tergoda

### 1. Proyek End-to-End
Bukan cloning tutorial! Buat aplikasi yang:
- Punya **database nyata**
- **Deploy ke cloud** (Vercel, Railway, atau AWS)
- Punya **user authentication**
- **Responsive** di mobile dan desktop
- **Open source** di GitHub

### 2. Dokumentasi yang Baik
README setiap proyek harus berisi:
- Deskripsi singkat (apa yang dibuat dan kenapa)
- Tech stack yang dipakai
- Fitur-fitur utama
- Screenshot / GIF demo
- Link demo (live)
- Cara install & run
- Jika tugas akhir/sekolah, sebutkan

### 3. Tunjukkan Proses
Jangan cuma hasil akhir. Tampilkan:
- **Commit history** yang menunjukkan progres
- **Issue tracking** — bagaimana kamu memecahkan masalah
- **Design decision** — kenapa pilih tech stack tertentu

## Checklist Sebelum Apply Kerja

✅ GitHub rapi dengan pinned proyek terbaik
✅ README detail untuk setiap proyek
✅ Link portofolio di CV dan LinkedIn
✅ Semua proyek bisa diakses (gak broken link)
✅ Tanpa typo dan kode berantakan
