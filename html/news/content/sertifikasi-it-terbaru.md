Sertifikasi IT bisa jadi **tiket emas** untuk fresh graduate yang ingin bersaing di pasar kerja. Sertifikasi membuktikan bahwa kamu punya standar kompetensi yang diakui secara global.

## Sertifikasi Cloud Computing

### AWS Certified Cloud Practitioner
- **Level:** Pemula
- **Biaya:** ~$100
- **Cocok untuk:** Semua role IT
- **Apa yang dipelajari:** Dasar-dasar AWS, layanan cloud, pricing, security
- **Gaji rata-rata:** Rp 8-15 juta/bulan (dengan sertifikasi)

### Google Cloud Associate Engineer
- **Level:** Intermediate
- **Biaya:** ~$125
- **Cocok untuk:** Backend dan DevOps
- **Duration persiapan:** 2-3 bulan belajar

### Microsoft Azure Fundamentals (AZ-900)
- **Level:** Pemula
- **Biaya:** ~$99
- **Frequently discounted** — cek Microsoft Learn untuk free voucher

## Sertifikasi Cybersecurity

### CompTIA Security+
- **Level:** Pemula
- **Diakui:** Global, termasuk Indonesia
- **Cocok untuk:** Semua yang tertarik cybersecurity
- **Masa berlaku:** 3 tahun

### Certified Ethical Hacker (CEH)
- **Level:** Intermediate
- **Biaya:** ~$1,199 (termasuk exam)
- **Cocok untuk:** yang mau jadi penetration tester

## Sertifikasi AI & Data

### TensorFlow Developer Certificate
- **Level:** Intermediate
- **Biaya:** ~$70
- **Cocok untuk:** AI/ML engineer
- **Format:** hands-on coding exam (5 jam)

### Google Data Analytics Professional Certificate
- **Level:** Pemula
- **Biaya:** ~$49/bulan (Coursera)
- **Cocok untuk:** Fresh graduate yang minat data

## Sertifikasi Web & Mobile

### Meta Front-End Developer Certificate
- **Level:** Pemula-Intermediate
- **Biaya:** ~$49/bulan (Coursera)
- **Cocok untuk:** Web developer

## Tips Mendapatkan Sertifikasi

1. **Manfaatkan diskon** — AWS dan Azure sering kasih diskon untuk mahasiswa
2. **Gunakan free resources** — YouTube, dokumentasi resmi, dan practice test gratis
3. **Sertifikasi ≠ garansi kerja** — tetap butuh portofolio dan skill praktis
4. **Pilih yang relevan** — jangan ambil semua, fokus ke jalur karir yang kamu incar
