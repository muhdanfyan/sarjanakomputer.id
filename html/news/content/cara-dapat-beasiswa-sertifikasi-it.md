Sertifikasi IT bisa mahal. Tapi tenang, banyak **program beasiswa voucher gratis** yang bisa kamu manfaatkan sebagai mahasiswa.

## 1. GitHub Student Developer Pack
**GRATIS!** Daftar pakai email `.ac.id`:
- AWS Educate — $100 credit
- DigitalOcean — $200 credit
- Namecheap — 1 tahun domain `.me` gratis
- JetBrains — IDE gratis selama jadi mahasiswa
- Heroku — $13 credit/bulan
- Dan 100+ benefit lainnya

👉 Daftar di: `education.github.com/pack`

## 2. Dicoding Indonesia
Program beasiswa rutin dari Dicoding bekerja sama dengan:
- **Kampus Merdeka** — sertifikasi gratis + proyek akhir
- **IDCamp** — belajar coding gratis dengan sertifikat
- **AWS re/Start** — untuk calon cloud practitioner

Cek jadwal pendaftaran di: `dicoding.com/beasiswa`

## 3. Microsoft Learn Student Ambassadors
Jadi Microsoft Learn Student Ambassador:
- Akses gratis ke sertifikasi Microsoft (AZ-900, DP-900, dll)
- Voucher exam gratis
- Azure credit $100/bulan
- Kesempatan ngajar dan networking

## 4. Google Cloud Skills Boost
- **30 hari free** akses quest & lab
- Sering ada **free voucher challenge** (selesaikan lab, dapatkan sertifikasi gratis)
- Google Developer Student Clubs (GDSC) — komunitas kampus dengan akses event eksklusif

## 5. AWS re/Start & AWS Educate
- **AWS Educate** — free learning path + $150 credit
- **AWS re/Start** — program pelatihan 12 minggu untuk yang gak punya latar belakang IT
- **AWS Community Day** — event gratis dengan workshop

## 6. Coursera for Campus
Banyak kampus yang kerja sama dengan Coursera. Cek apakah kampusmu menyediakan:
- **Coursera for Campus** — akses gratis ke ribuan kursus
- **Google Career Certificates** — termasuk kursus IT support, data analytics, UX design

## Tips Mengoptimalkan Voucher

1. **Jangan ambil semua sekaligus** — fokus 1-2 sertifikasi yang relevan
2. **Gunakan masa trial** — banyak platform kasih 7-30 hari gratis
3. **Ikuti event challenge** — Google dan AWS sering adain challenge dengan reward voucher
4. **Manfaatkan komunitas** — grup Telegram/Discord sering bagi info beasiswa
